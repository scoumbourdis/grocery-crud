Grocery CRUD
=============
Grocery CRUD is a PHP Codeigniter Framework library that creates a full functional CRUD system without the requirement of extra customisation to the JavaScripts or the CSS to do it so.

Grocery CRUD 是一套PHP Codeigniter Framework 的 library，可以快速建置出完整CRUD功能的套件（Create、Read、Update、Delete，新增、讀取、更新、刪除），而且不再需要額外的JavaScripts或CSS即可快速達成！

官方網站： http://www.grocerycrud.com

官方版本中文僅有簡體中文，本專案版本為簡繁並存，並已經預設為"chinese-trad"正體中文（原本的簡體中文更名為chinese-simp）。
